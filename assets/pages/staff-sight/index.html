Staff-Sight

Description

Staff-Sight is a monitoring application designed to track employee activity on company-owned devices. The application logs keyboard activity, running processes, and system information, providing valuable insights into employee productivity and potential security threats.

Features

  • Keylogging: Staff-Sight logs all keyboard activity, including keystrokes and special keys.
  • Process Monitoring: Staff-Sight monitors instances of suspicious activity.
  • System Information: Staff-Sight collects system information, including CPU usage, RAM usage, and system time.
  • Encryption: All log data is encrypted using Fernet encryption.
  • Email Notifications: The application sends encrypted log files and the master key to a designated administrator email account.
